# Report Slide Structure — Oblique Performance Marketing Report

This is the canonical slide-by-slide spec for all Oblique performance reports. Based on the Petico bi-weekly report format. Follow this structure for every client unless Sean explicitly requests changes.

---

## Slide 1 — Title Slide (dark)

**Layout:** Dark navy background (#1A1A2E), centred content
**Content:**
- Tag line (small, accent colour): "Performance Marketing"
- Sub-tag: "Bi-Weekly Reporting" or "Monthly Reporting" (match cadence)
- Date range: e.g. "1–15 March 2026"
- "By Oblique" (small, bottom)
- "Tracker: [link]" (bottom, if tracker URL is provided)

---

## Slide 2 — Summary: Previous Period Ad Performance

**Layout:** Light background, two-column (left: visual/chart space, right: key insights bullets)
**Title:** "Summary Ad Performance [PREVIOUS PERIOD]"

**Content — Metrics snapshot:**
| Metric | Value |
|--------|-------|
| ROAS | X.XX |
| CPP | RM/$/X |
| AOV | RM/$/X |
| Total Spend | RM/$/X |
| Purchases | X |
| CVR | X.XX% |

**Key Insights (3–4 bullets max):**
Write punchy, strategic sentences explaining WHAT happened and WHY. Examples:
- "ROAS improved from X to Y — driven by [campaign change / creative rotation / audience refinement]"
- "CPP decreased despite stable spend — [awareness campaign efficiency / better audience targeting]"
- "AOV lifted to RM X — [new customer mix / premium product push]"

---

## Slide 3 — Summary: Current Period Ad Performance

**Same layout as Slide 2**, but for the current (reporting) period.

**Title:** "Summary Ad Performance [CURRENT PERIOD]"

Show the SAME metrics. Add delta indicators vs previous period:
- ↑ in green if improved (ROAS ↑, CPP ↓, CVR ↑, AOV ↑)
- ↓ in red if worsened

**Key Insights:** What changed vs last period? Any notable anomalies?

---

## Slide 4 — Section Divider: Conversion Performance

**Layout:** Full dark navy background
**Content:** Large white text — "Conversion Performance"
- Optional: accent line or subtle icon (shopping cart, etc.)

---

## Slide 5 — Conversion: Ad Platform vs Backend Actual

**Layout:** Light background with two sub-panels or a comparison table
**Title:** "Conversion Campaign | Ad Platform vs Backend Actual"

**Content:**
Two columns or a chart:
- Column A: "Ad Platform Reported" (Windsor/Meta/Google data)
  - Total Revenue (attributed): RM/$ X
  - ROAS (platform): X.XX
- Column B: "Backend Actual" (Shopify/actual orders)
  - Total Revenue (actual): RM/$ X
  - Run Rate for month: RM/$ X (if mid-month)

**Key Insights:**
Explain the gap between platform-reported and backend actual (this is normal due to attribution windows, view-through conversions, etc.).

Note: If backend data was not provided by Sean, show Column B as "TBC — awaiting backend data" and note it for follow-up.

---

## Slide 6 — P&L Analysis (Ads Only)

**Layout:** Light background, chart + table + insights
**Title:** "Profit and Loss Analysis — Ads ONLY | [Margin %] margin"

**Content:**
Table showing:
| Month | Spend | Revenue | Gross Profit | Net Profit (Ads) | ROAS |
|-------|-------|---------|-------------|-----------------|------|
| [previous months for context] | | | | | |
| [current period] | | | | | |

Calculate:
- Gross Profit = Revenue × margin%
- Net Profit = Gross Profit − Spend
- Break-even ROAS = 1 / margin% (e.g. 18% → 5.56)

**Key callouts:**
- Highlight whether the current period is net positive or negative
- State the break-even ROAS threshold clearly

## Slide 11 — Campaign Structure Overview

## Slide 12 — New vs Returning Breakdown (Meta, if data available)

**Layout:** Side-by-side comparison: previous period vs current period
**Title:** "Conversion Campaign | Meta Ads — New vs Returning Customer"

Show:
- New customer CPP, ROAS, % of spend
- Returning/existing customer CPP, ROAS, % of spend

Key Insight: explain the strategic decision (e.g. why Oblique shifted budget toward remarketing, what impact it had).

## Slide 13+ — Top Creatives: Conversion (Segment 1)

**Layout:** Creative spotlight cards — 3 per slide
**Title:** "Conversion Campaign | Meta Ads Top Creative — [Segment, e.g. DOG / CAT / General]"

**Each card (#1, #2, #3 ranked by purchases):**
- Source (ad name from Windsor)
- Rank label: "#1", "#2", "#3"
- Metrics:
  - Purchases
  - ATC (Add to Cart)
  - CPP
  - CTR
- Ad name / description of creative angle

**Insights section** (bottom of slide):
- Explain WHAT creative angle is winning (e.g. "Value-driven messaging outperforms lifestyle imagery")
- Explain WHY it's working (psychology, audience insight)

**Recommendation section:**
- 1–2 concrete next steps based on what's winning

*Note at bottom: "Visual is analyzed independently. Ads loaded into multiple campaigns — total results may be higher than stated."*

## Notes on data presentation

- **Always use local currency** — RM for Malaysia, SGD/$ for Singapore. Confirm with Sean if unsure.
- **Round sensibly** — ROAS to 2 decimal places, CPP to whole number, AOV to whole number, CTR to 2 decimal places as a %
- **Delta arrows:** Use ↑ and ↓ with colour. Green = good direction (ROAS ↑, CPP ↓, CVR ↑). Red = bad direction.
- **Never leave empty cells** — if data is unavailable, write "—" or "N/A"
- **Disclaimer on creative data** (standard): include on every creative slide — *"Please note that visual is analyzed independently. Ads are loaded into multiple campaigns so total results might be higher than stated here."*
